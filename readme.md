# UML (Lenguaje de modelado unificado)

Permite:

* Visualizar
* Especificar
* Construir y
* Documentar

los artifactos de un sistema de software.

El vocabularío de UML incluye 3 tipos de bloques de construcción:

1. Elementos
2. Relaciones
3. Diagramas
